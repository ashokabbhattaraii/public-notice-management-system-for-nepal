================================================================================
                        CUSTOMIZATION QUICK GUIDE
================================================================================

Change App Colors (30 seconds):
  1. Open app/globals.css
  2. Find "--primary: oklch(0.55 0.15 265)"
  3. Change "265" to any hue 0-360
  4. Refresh page to see change

Add a Notice (2 minutes):
  1. Open lib/mock-data.ts
  2. Find MOCK_NOTICES array
  3. Copy last notice object
  4. Modify: id, title, description, category, content
  5. Save file - notice appears on home page

Change App Name (1 minute):
  1. Open components/layout/header.tsx
  2. Change "NoticeBoard" text
  3. Open app/layout.tsx
  4. Change metadata title
  5. Save and refresh

Update Homepage Text (1 minute):
  1. Open app/page.tsx
  2. Find "Institutional Notices & Updates"
  3. Change to your text
  4. Modify description below
  5. Save and refresh
