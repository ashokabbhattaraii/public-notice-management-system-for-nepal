## 🎨 Customization Quick Guide

### Change Brand Colors (2 minutes)
```css
/* Open app/globals.css, change this line: */

```

### Add New Notice (1 minute)
```typescript
/* Open lib/mock-data.ts, add to MOCK_NOTICES: */
{
  id: 'your-id',
  title: 'Your Notice Title',
  description: 'Brief description',
  category: 'exams', // or other category
  content: 'Full content here',

}
```

## 🆘 Troubleshooting

### "npm/pnpm error"
```bash
rm -rf node_modules pnpm-lock.yaml
pnpm install
pnpm dev
```

### "Port already in use"
```bash
# Kill process on port 3000
# macOS/Linux: lsof -ti :3000 | xargs kill -9
# Windows: netstat -ano | findstr :3000
```

### "Changes not showing"
```bash
# Clear next cache
rm -rf .next
pnpm dev
```

### "Styles not loading"
```bash
# Clear Tailwind cache
pnpm build
pnpm dev
```
