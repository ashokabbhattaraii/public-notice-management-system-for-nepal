## 💡 Interactive Features You'll Experience

### Expand/Collapse Filters
- Click filter header → Section expands/collapses smoothly
- Saves space on screen
- Smooth 200ms animation

### Click to Filter
- Click category card in stats → Auto-filters instantly
- Results update in real-time
- No page reload

### Add/Remove Filters
- Click any filter option → Instantly applied
- Active filter shows as chip below
- Click chip's X to remove individual filters

### Search in Real-Time
- Type in search box → Results update as you type
- No delay, instant filtering
- Shows "No results" if nothing matches

### Hover Effects
- Hover over notice card → Grows slightly, shadow increases
- Hover over filter → Subtle background highlight
- Icons scale up when hovering
- Color transitions smoothly

### Interactive Deadlines
- Notice shows "Deadline in 5 days"
- Auto-calculated from deadline date
- Shows "⚡ Deadline Soon" for <7 days
- Updates relative to current date

## 👆 User Interactions

### Browsing Notices
1. Page loads with all notices shown
2. Stats section immediately visible
3. User sees featured high-priority notice
4. Scrolls to see more notices

### Filtering by Category
1. User clicks category card (e.g., "Exams")
2. Stats card shows highlight
3. Notices instantly filter to only Exams (3 results)
4. Notice count updates to "3"
5. Sidebar shows applied filter as chip

### Searching
1. User types "PSC" in search box
2. Results filter in real-time
3. Only notice with "PSC" in title/content shown
4. Sidebar shows search chip: [Search: "PSC" ×]

### Complex Filtering
1. User selects "Exams" category
2. Then selects "High Priority"
3. Results show intersection (high priority exams only)
4. Both filters shown as removable chips
5. Can click × on either chip to remove

### Clearing Filters
1. User clicks "Clear All Filters"
2. All filters instantly removed
3. All 10 notices shown again
4. Filter chips disappear
